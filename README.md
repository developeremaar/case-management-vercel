# نظام إدارة الحالات

نظام إدارة حالات (RTL) مخصص للمنظمات غير الربحية والجمعيات الخيرية.

## تشغيل المشروع محليًا / في Codespaces

- **الإصدار المطلوب من Node.js:** `20` أو أحدث.
- **الإصدار المطلوب من npm:** `10` أو أحدث.

```bash
npm ci
npm run build
npm run lint
```

> ملاحظة: قد يفشل `npm run lint` بسبب مشاكل lint موجودة مسبقًا وغير مرتبطة بهذا التغيير.
