# Supabase Current Schema

هذه الملفات تمثل النسخة الحالية المعتمدة من قاعدة بيانات Supabase.

يجب على أي وكيل برمجي أو مساهم في المشروع الاعتماد على هذه الملفات كمصدر الحقيقة عند مراجعة:
- الجداول
- الحقول
- العلاقات
- سياسات RLS

مهم:
- لا تستخدم أي عمود غير موجود في هذه الملفات.
- لا تفترض وجود أعمدة بناءً على أسماء مشابهة.
- لا تنفذ migrations إلا بعد موافقة صريحة.
- عند وجود تعارض بين هذه الملفات وأي تقرير سابق، يتم اعتماد هذه الملفات.
- يمنع استخدام spread مثل ...formData مع Supabase.
- يجب أن تكون عمليات insert/update بحقول صريحة فقط.

الملفات:
1. 01-tables-list.csv  
   قائمة الجداول الحالية.

2. 02-tables-columns-relations.csv  
   الجداول والحقول وأنواع البيانات والعلاقات.

3. 03-foreign-keys.csv  
   العلاقات بين الجداول.

4. 04-rls-policies.csv  
   سياسات RLS الحالية.
