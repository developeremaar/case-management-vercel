# Modules Roadmap

## الوحدات الحالية
- الجهات والهوية.
- العضويات والأدوار والصلاحيات.
- المستفيدون.
- الحالات وسير العمل.
- السجلات والمرفقات والإشعارات الأساسية.

## الوحدات الناقصة
- الاشتراكات والباقات (مطلوبة الآن).
- التكاملات المرنة لكل جمعية حسب المزود (WhatsApp/SMS/Email/Domain/Webhooks) (مطلوبة الآن).
- محرك إشعارات موسع (مطلوب الآن) لدعم الموظفين والمستفيدين والحالات والأقسام والأدوار وتطبيق الجوال لاحقًا.
- بوابة خارجية مستقبلية.

## ترتيب التنفيذ المقترح
1. سد فجوات العزل (RLS/اختبارات).
2. وحدة الاشتراكات والباقات.
3. وحدة التكاملات.
4. وحدة الإشعارات.
5. API خارجي مؤمن ومقنن.
6. البوابة الخارجية.

## ما ينفذ الآن
- الاشتراكات والباقات.
- التكاملات.
- الإشعارات.
- مواصفات API الداخلي والخارجي.

## ما يمكن تأجيله
- تطبيق الجوال كاملًا.
- واجهات البوابة الخارجية النهائية.
